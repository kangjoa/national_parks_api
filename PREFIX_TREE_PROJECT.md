# Trees Project

## How to run tests

Run the ported tests, `prefix-tree-node.js`, based on [prefixtree_test.py](https://github.com/Tech-at-DU/ACS-3110-Trees-Sorting/blob/master/Code/prefixtree_test.py)

```zsh
cd server/tests && npm test prefix-tree-node
```

Run the new unit tests for prefix-tree.js:

```zsh
cd server/tests && npm test prefix-tree
```
